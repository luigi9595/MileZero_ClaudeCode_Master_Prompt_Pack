# PHYSICS_TEST_PLAN

## Goals
- verify drivability
- verify surface differentiation
- verify vehicle differentiation
- verify damage consequences

## Core tests
1. Spawn and drive smoke test
2. Dry surface baseline loop
3. Gravel comparison loop
4. Wet comparison loop
5. Coupe agility loop
6. Hatch stability loop
7. Pickup bump/roll loop
8. Light crash consequence test
9. Medium crash consequence test
10. Heavy crash consequence test

## Logging rule
Every meaningful test should produce:
- date/time
- vehicle
- surface
- objective
- observed behavior
- pass/fail or partial
- next tuning action
