# PROJECT_VISION

MileZero is an original single-player driving sandbox focused on believable vehicle behavior, compact free-roam exploration, and meaningful crash consequences.

The first vertical slice should feel like a serious indie driving simulator foundation:
- immediate drivable fun
- clearly different vehicle handling archetypes
- surfaces that materially change traction and braking
- crashes that alter drivability
- compact but replayable route design
- clean architecture that supports continued development
